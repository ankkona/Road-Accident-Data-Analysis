# Road-Accident-Data-Analysis
This Road Accident Dashboard is an interactive analytical dashboard built entirely using Microsoft Excel to analyze road accident casualty data.
The dashboard provides a clear, visual summary of total casualties, severity levels, vehicle types involved, road conditions, and environmental factors such as light and road surface.

# Tools Used
- MS Excel
- Pivot Tables
- Pivot Charts(Donut & Pie)
- Slicers & Timelines
- Conditional Formatting

# Dataset used
- <a href="https://github.com/ankkona/Road-Accident-Data/blob/main/Road%20Accident%20Data.xlsb">Road Accident Data</a>

# Questions(KPIs)  
- What is the total number of casualties?

- How many casualties are fatal, serious, or slight?

- Which vehicle type causes the highest number of casualties?

- How do current year casualties compare with previous year trends?

- Which road type is most accident-prone?

- Do accidents occur more in urban or rural areas?

- How do light conditions (daylight vs dark) affect accidents?

- What impact does road surface condition (dry, wet, snow/ice) have on casualties?

- How do casualties vary month-wise across years?
